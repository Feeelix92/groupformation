Group Formations Mockups

Mockups Version 1.0

Bei dieser Version soll eine einheitliche und benutzerfreundliche Struktur der Dozenteneinstellungen sowie Fragebögen für Studierende erarbeitet werden. Der Fokus ist auf die passende bzw richtige Wahl der Werkzeuge zur Einstellung der Optionen, sowie die Volständigkeit aller Inhalte und Anforderungen gelegt. Auf die Farben wird bewusst verzichtet. Es wird ein soggenanntes "Responsive Design" implementiert, dass eine optimale Anzeige auf allen herkömmlichen Displaygrößen gewährleistet.
